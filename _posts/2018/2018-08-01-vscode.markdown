## Remote SSH Configuration

- Configuration `~/.ssh/config`

```bash
Host <HOSTNAME>
    HostName <IP>
    User <USER_NAME>
    Port 22
```

- To enable Python module clicking, go to extensions -> click `"install on <HOST>"` on python extension (10-15MB, typically less than 1-2%)
