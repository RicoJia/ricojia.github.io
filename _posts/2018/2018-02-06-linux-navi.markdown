## About Navi

[Navi](https://github.com/denisidoro/navi) is an interactive command-line cheatsheet tool that helps you find, fill in, and run common commands without memorizing every flag or workflow. It lets you browse cheatsheets, select commands, and dynamically choose argument values before execution.

Some navi commands that I use are:

| Shortcut          | What it does                                                                                   |
| ----------------- | ---------------------------------------------------------------------------------------------- |
| `Ctrl-G`          | Source / build workflow                                                                        |
| `Ctrl-F, M`       | Node commands                                                                                  |
| `Ctrl-F, V`       | Select a service, then use `Ctrl-C`                                                            |
| `Ctrl-F, I`       | Actions                                                                                        |
| `Right Ctrl-F, ,` | Topic echo check — needs confirmation whether right Ctrl is required or if left Ctrl works too |
